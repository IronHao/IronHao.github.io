左值、右值与引用

左值(lvalue)&右值(rvalue)

An lvalue is an expression that identifies a non-temporary object.

An rvalue is an expression that identifies a temporary object or is a value (such as a literal constant) not associated with any object.

  通俗的讲，左值就是我们定义的有名字的非临时的变量(全局变量？)。右值则是一些声明在函数里的临时变量或者没有名字的值。可以说，使用取地址符&对某个值取地址，如果不能得到地址，则是右值！

int y = 2; // 'y' is lvalue
int x = 2; // '2' is rvalue
int z = x + y; // 'x + y' is rvalue, 'z' is lvalue

// 下面这个函数中的'x','y','tmp'都是右值，可以用&去判断，编译器会报错
void swap( vector<string> & x, vector<string> & y ) 
{
    vector<string> tmp = x;
    x = y;
    y = tmp;
}
